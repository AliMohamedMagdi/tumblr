# Tumblr mini

> Full-stack Tumblr clone (React + Apollo)

## Install

1. Clone project

```bash
git clone https://github.com/shwilliam/mini-tumblr-clone
```

2. Start server

```bash
cd server

npm i

npm start
```

3. With the server running, start the dev client in a new terminal window

```bash
cd client

npm i

npm start
```
